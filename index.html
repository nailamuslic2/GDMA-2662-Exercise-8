Fluid Layout

In a fluid layout, the page area and its grid resize proportionally to fill the available width of the screen or window. That is easily accomplished with percentages! You can also rely on flex, without a width, or in the future CSS Grid when it's more widely supported.

Flexible Images

Every now and then a solutio is simple. Take, for example, the styles required to make images scal to fit the size of their container: max-width: 100%. That's it!

Media Queries

Media queries apply different styles based on characteristics of the browser: its width, whether it is vertically or horizontally oriented, its resolution, and more. They are what make it possible to send a one-column layout to small screens and a multicolumn layout to larger screens on the fly.

works

A breakpoint is the point at which we use a media query to introduce a style change. We tend to base our breakpoints on the common device sizes.

Mobile-first is when we start by writing our CSS for mobile devices and then use media queries to add in styling for larger screen sizes. When we style for desktop, we're adding widths, columns, and moving things around. We're adding complexity. Mobile layouts tend to be very simple, making it very easy to start there.
